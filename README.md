# minimal-java-agent

A minimal template built with **Java 21** and **Spring Boot (latest)**, designed to create modern agents that handle and orchestrate AI requests.

## 🚀 Features
- **Spring Boot latest**: fast startup and minimal configuration.
- **Java 21 virtual threads**: lightweight concurrency for scalable services.
- **REST Controller skeleton**: entry point for agent requests.
- **Health check**: `/actuator/health` ready for orchestration tools.
- **Docker-ready**: containerized for immediate deployment.
